# 🐝 BookBee

BookBee is a full-stack ticket booking platform that allows users to seamlessly browse movies, select seats, and book tickets online. It features a robust user interface for customers and a dedicated admin dashboard for managing theaters, shows, and promotions.

## ✨ Features

- **User Authentication:** Secure login and registration.
- **Movie & Theater Browsing:** View currently running movies, trailers, and theater locations.
- **Interactive Seat Mapping:** Real-time seat layout selection.
- **Secure Checkout:** Streamlined ticket booking and payment success flow.
- **Admin Dashboard:** Manage movie releases, theater layouts, promotions, and verify tickets.

## 🛠️ Tech Stack

**Frontend (Client):**

- React (via Vite)
- CSS / UI Styling

**Backend (Server):**

- Node.js & Express
- MongoDB (Mongoose models)
- Supabase (Database/Storage)
